用户管理接口

提供用户清单、注册用户、修改用户信息等。

一、接口如下：

1. 用户清单：/api/userManger/getList
2. 注册用户：/api/userManger/create ：
  • 变量email 邮箱账户，要保证唯一性
  • 变量password 密码长度6-18位
