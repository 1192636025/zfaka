找回密码流程：

1.访问地址：http://xiuxiuhou.com/member/forgetpwd


2.ajax提交，发送邮件

接口地址：  /member/forgetpwd/ajax/

参数：
email  邮箱
vercode   图形验证码
csrf_token token

返回：
json格式  code,msg

注册成功 code 1       成功后，请显示对应 模版页面中的文字信息

注册失败 code 1000-1002



3.客户收到验证邮件后，点击链接，跳到 http://xiuxiuhou.com/member/forgetpwd/reset


4.重设密码接口
接口地址：  /member/forgetpwd/resetajax/

参数：
email  邮箱   hidden
code   邮箱验证码  hidden
password   密码
csrf_token token

返回：
json格式  code,msg

注册成功 code 1       成功后，跳转到登录

注册失败 code 1000-1002
